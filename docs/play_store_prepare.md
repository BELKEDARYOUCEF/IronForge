## Full Description Draft

IronForge is an offline-first workout logger for strength training. Create routines, log sets, track volume, save custom exercises, and follow progress charts without needing an account or cloud sync.

Current features:

- offline workout logging;
- local workout history;
- progress charts;
- editable routines;
- custom exercises and favorites;
- onboarding preferences;
- kg/lbs workout entry.
- rest timer;
- plate calculator;
- premium red/black IronForge UI.
